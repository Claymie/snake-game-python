# 🐍 Python Red-Head Snake
Классическая игра "Змейка" с кастомным дизайном и улучшенной механикой управления.

## 🚀 Особенности
- **Pro Navigation**: Голова змеи выделена контрастным цветом (бордовым) для удобства ориентирования.
- **Score System**: Встроенный счетчик очков в реальном времени.
- **Adjustable Gameplay**: Оптимизированная скорость и увеличенное игровое поле (800x600) для комфортной игры.
- **Collision Logic**: Проработанная система обработки столкновений со стенками и собственным телом.

## 🛠 Технологии
- **Python 3.x**
- **Pygame**: Библиотека для отрисовки графики и обработки событий.

## 🕹 Как запустить
1. Установите pygame: `pip install pygame`
2. Запустите файл: `snake.py`
